نصب وابستگی‌ها
تمامی وابستگی‌های موجود در این کد عبارتند از:
cv2 (OpenCV)
numpy
pandas
fastapi
easyocr
uvicorn
rapidfuzz (اختیاری)
برای نصب این وابستگی‌ها می‌توانید از دستور زیر استفاده کنید:

pip install opencv-python-headless numpy pandas fastapi easyocr uvicorn
نصب rapidfuzz (اختیاری):
در صورتی که تمایل دارید از قابلیت fuzzy matching استفاده کنید (که در کد برای تطابق لیبل‌ها استفاده شده)، می‌توانید rapidfuzz را نصب کنید:

pip install rapidfuzz
3. نصب OpenCV
اگر از نسخه‌ی کامل OpenCV (با GUI) استفاده می‌کنید، می‌توانید از opencv-python استفاده کنید. در غیر اینصورت، برای کار بدون نیاز به GUI از opencv-python-headless استفاده کنید:

pip install opencv-python-headless
4. نصب وابستگی‌های دیگر (در صورت نیاز به دیگر ابزارها)
اگر نیاز به ابزارهای بیشتری دارید (مثلاً برای پشتیبانی از شبکه‌های GPU برای EasyOCR)، می‌توانید از دستور زیر برای نصب آن‌ها استفاده کنید:

pip install torch torchvision torchaudio
5. تنظیم HTTPS
برای اجرای سرور با HTTPS، نیاز به فایل‌های SSL Certificate دارید. شما باید دو فایل key.pem و cert.pem (که برای HTTPS استفاده می‌شوند) را در همان پوشه‌ای که اسکریپت را اجرا می‌کنید قرار دهید. اگر از گواهینامه‌های SSL خودتان استفاده نمی‌کنید، می‌توانید از گواهینامه‌های آزمایشی مانند Let's Encrypt استفاده کنید.

اگر از HTTPS استفاده می‌کنید، سرور روی پورت 8443 اجرا خواهد شد و می‌توانید به API از طریق آدرس https://localhost:8443 دسترسی پیدا کنید.
